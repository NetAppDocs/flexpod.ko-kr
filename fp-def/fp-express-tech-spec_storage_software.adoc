FlexPod Express는 NetApp ONTAP 운영 체제를 실행하는 스토리지 시스템을 지원합니다.



== NetApp ONTAP를 참조하십시오

NetApp ONTAP 소프트웨어는 AFF, ASA 및 FAS 스토리지 시스템에서 실행되는 운영 체제입니다. ONTAP는 중단 없는 운영, 중단 없는 업그레이드 및 민첩한 데이터 인프라를 지원하는 확장성이 뛰어난 스토리지 아키텍처를 제공합니다.

ONTAP에 대한 자세한 내용은 를 참조하십시오 http://www.netapp.com/us/products/platform-os/ontap/index.aspx["ONTAP 제품 페이지"^].
