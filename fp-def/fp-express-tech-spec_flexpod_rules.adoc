= FlexPod 규칙
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


[role="lead"]
FlexPod 설계는 다양한 하드웨어 구성 요소와 소프트웨어 버전을 포괄하는 유연한 인프라 아키텍처를 지원합니다.

이 규칙 세트를 유효한 FlexPod 구성을 구축하거나 조합하기 위한 지침으로 사용하십시오. 이 문서에 나열된 숫자와 규칙은 FlexPod의 최소 요구 사항을 나타내며, 다양한 환경 및 사용 사례에 따라 포함된 제품군 내의 지원 제품을 사용하여 확장할 수 있습니다.
