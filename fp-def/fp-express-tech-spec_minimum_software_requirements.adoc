== FlexPod Express with NetApp AFF 또는 FAS의 소프트웨어 요구 사항

FlexPod Express with NetApp FAS의 소프트웨어 요구 사항은 다음과 같습니다.

* ONTAP 9.1 이상
* Cisco NX-OS 버전 7.0(3)I6(1) 이상
* Cisco UCS 관리 구성에서 Cisco UCS Manager UCS 4.0(1b)


모든 소프트웨어는 목록에 포함되어야 하며 지원되어야 합니다 http://mysupport.netapp.com/matrix["NetApp IMT를 참조하십시오"^]. 특정 소프트웨어 기능은 이전 아키텍처에서 요구하는 최소 사양보다 더 최신 버전의 코드를 필요로 할 수 있습니다.
