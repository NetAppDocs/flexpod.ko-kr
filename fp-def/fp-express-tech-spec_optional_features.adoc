== iSCSI 부트 옵션

FlexPod 익스프레스 아키텍처는 iSCSI 부트를 사용합니다. iSCSI 부팅 옵션에 대한 최소 요구 사항은 다음과 같습니다.

* NetApp 스토리지 컨트롤러에서 활성화된 iSCSI 라이센스/기능
* NetApp 스토리지 컨트롤러 HA 쌍의 각 노드에 있는 2포트 10Gbps 이더넷 어댑터
* iSCSI 부팅이 가능한 Cisco UCS 서버의 어댑터
