= 기타 요구 사항
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


[role="lead"]
FlexPod 익스프레스에 대한 추가 요구 사항은 다음과 같습니다.

* VMware ESXi를 하이퍼바이저로 사용할 계획이라면 link:https://compatibilityguide.broadcom.com/["Broadcom Compatibility Guide"]을 참조하여 하드웨어 및 소프트웨어 구성 요소가 지원되는지 확인하십시오.
* OpenShift 기반 배포의 경우 OpenShift와 호환되는 필요한 하드웨어 및 소프트웨어 구성 요소가 있는지 확인하십시오. 자세한 내용은 link:https://catalog.redhat.com/en/search["OpenShift 카탈로그"]을 참조하십시오.
