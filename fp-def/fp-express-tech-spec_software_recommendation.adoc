= 소프트웨어 버전 권장 사항
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


[role="lead"]
이 섹션에서는 FlexPod Express에 권장되는 소프트웨어 버전에 대해 설명합니다.

* ONTAP 버전: 모든 FlexPod Express 구성에는 최신 ONTAP 버전을 사용하는 것이 좋습니다. 최신 ONTAP 버전은 link:https://docs.netapp.com/us-en/ontap/release-notes/index.html["ONTAP 릴리스 정보"^]을 참조하십시오.
* Cisco NX-OS 버전: Cisco 9000 시리즈 스위치에 대한 일반적인 Cisco NX-OS 버전 권장 사항은 link:https://www.cisco.com/c/en/us/td/docs/switches/datacenter/nexus9000/sw/recommended_release/b_Minimum_and_Recommended_Cisco_NX-OS_Releases_for_Cisco_Nexus_9000_Series_Switches.html["Cisco Nexus 9000 시리즈 스위치에 권장되는 Cisco NX-OS 릴리스"^]을 참조하십시오.
* Cisco UCS 펌웨어 버전:
+
** 최신 버전을 보려면 이 링크를 참조하세요. link:https://www.cisco.com/c/en/us/support/servers-unified-computing/intersight/products-release-notes-list.html["Cisco Intersight 관리형 모드 인프라 및 서버 펌웨어, 섀시 펌웨어 및 PCIe 노드 펌웨어"^]
** 최신 버전을 보려면 이 링크를 참조하세요. link:https://intersight.com/help/saas/supported_systems/Intersight_Standalone_Mode["Cisco Intersight Standalone Mode 및 CIMC 펌웨어"^]





NOTE: 모든 소프트웨어 구성 요소는  https://imt.netapp.com/imt/#welcome["NetApp IMT를 참조하십시오"^]에 나열되고 지원되어야 하며, 모든 NetApp 하드웨어 구성 요소 및 ONTAP 버전은  https://hwu.netapp.com/Home/Index["NetApp Hardware Universe를 참조하십시오"^]에 나열되고 지원되어야 하고, 모든 Cisco 하드웨어 구성 요소는  https://ucshcltool.cloudapps.cisco.com/public/["Cisco HCL"^]에 나열되고 지원되어야 합니다.
