= 지원되는 구성과 검증된 FlexPod 구성 비교
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


[role="lead"]
FlexPod 아키텍처는 이 문서에 설명된 규칙 집합에 의해 정의됩니다. 하드웨어 구성 요소와 소프트웨어 구성은  https://ucshcltool.cloudapps.cisco.com/public/hcl["Cisco HCL(하드웨어 호환 목록)"^] 및  http://mysupport.netapp.com/matrix["NetApp 상호 운용성 매트릭스 툴(IMT)"^]에서 지원되어야 합니다.

각 Cisco 검증 설계(CVD) 또는 NetApp 검증 아키텍처(NVA)는 검증된 FlexPod 구성을 나타냅니다. Cisco와 NetApp은 이러한 구성 조합을 문서화하고 광범위한 엔드투엔드 테스트를 통해 검증합니다. 이러한 구성과 다른 FlexPod 배포도 이 문서의 지침을 준수하고 모든 구성 요소가 Cisco HCL 및 NetApp IMT에 호환 가능한 것으로 나열되어 있는 경우 완전히 지원됩니다.

예를 들어, 소프트웨어, 하드웨어 및 구성이 이 문서에 정의된 지침을 충족하는 경우 스토리지 컨트롤러 또는 Cisco UCS 서버를 추가하고 소프트웨어를 최신 버전으로 업그레이드할 수 있습니다.
