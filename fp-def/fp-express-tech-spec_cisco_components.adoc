== Cisco UCS X-Series Direct

Cisco UCS X-Series Direct는 Cisco UCS Fabric Interconnect 9108 100G 한 쌍과 Cisco UCS X9508 섀시 및 Cisco UCS X-Series 컴퓨팅 노드를 통합하여 UCS X-Series Direct 솔루션을 제공하는 새로운 아키텍처입니다.

Cisco UCS X-Series Direct는 다음 구성 요소로 구성됩니다.

* Cisco UCS 9108 100G 패브릭 인터커넥트
* Cisco UCS X9508 섀시
* Cisco UCS X-Series 컴퓨팅 노드
* 선택적 구성 요소: PCIe 노드 및 X-Fabric 모듈

== Cisco UCS C-Series 랙 서버 옵션

Cisco UCS C-시리즈 랙 서버는 1U 및 2U 랙 유닛(RU) 구성으로 제공되며, 다양한 CPU, 메모리 및 I/O 옵션을 선택할 수 있습니다. 다음 표에 나열된 부품 번호는 기본 서버용입니다. CPU, 메모리, 드라이브, PCIe(Peripheral Component Interconnect Express) 카드는 포함되지 않습니다. 여러 구성 옵션을 사용할 수 있으며 FlexPod 아키텍처에서 지원됩니다.
