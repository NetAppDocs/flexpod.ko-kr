== NetApp AFF/ ASA/ FAS와 함께 사용하는 FlexPod Express의 연결 요구 사항

FlexPod Express와 NetApp AFF/ ASA/ FAS의 연결 요구 사항은 일반적으로 다음과 같습니다.

* NetApp 스토리지 컨트롤러는 Cisco Nexus 스위치에 직접 연결해야 합니다.
* Cisco UCS X-Series Direct 구축의 경우 NetApp 스토리지 컨트롤러를 내장된 패브릭 인터커넥트에 직접 연결할 수 있습니다.
* 핵심 FlexPod 구성 요소 사이에 추가 장비를 인라인으로 배치할 수 없습니다.
* Cisco Nexus 9000 시리즈 스위치를 NetApp 스토리지 컨트롤러에 연결하려면 가상 포트 채널(vPC)이 필요합니다.
* 반드시 필요한 것은 아니지만 환경 전체에서 점보 프레임 지원을 사용하도록 설정하는 것이 좋습니다.
