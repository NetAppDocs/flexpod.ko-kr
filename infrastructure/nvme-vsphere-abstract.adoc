= 추상화
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./../media/


이 기술 보고서에서는 FlexPod 기반의 VMware vSphere 7을 사용한 FC-NVMe(NVMe over Fibre Channel) 성능에 대한 개괄적인 개요를 제공합니다. FlexPod는 애플리케이션에 고성능 스토리지 액세스를 제공하기 위해 FC-NVMe를 구축하기 위한 최적의 공유 인프라 기반입니다.
